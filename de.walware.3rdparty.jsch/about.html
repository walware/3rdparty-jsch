Encryption Items

The plug-in contains the implementation of a keyed-hash message authentication code (HMAC). It also relies on cryptographic algorithms provided by the Java™ Cryptography Extension (JCE) to support the implementation of the Secure Shell 2 (SSH2) protocol.

The following Java classes contain encryption items (the plug-in may contain both binary and source code):

  • com.jcraft.jsch.jcraft.HMAC
